This is my attempt at recreating Pong, in an effort to learn how to code. <br><br>
The "W" and "S" keys control the left paddle, and the arrow keys for up and down control the right paddle. It is an endless, local multiplayer game.<br><br>
If the ball's momentum slows too much, a burst of speed in a random direction will be applied. Perhaps more speed than there should be, but I personally thought it was funny.
